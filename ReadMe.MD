# Trident Compiler

An experimental compiler for the Trident language.

## Language Specs
Trident language [Scpec](https://github.com/parixitsapkota/trident/blob/main/docs/spec.md)

## Build & Run

- Build Requirements : `clang-format clang make`

```bash
make MODE=release
```

- Run

```bash
./trident <FILE>
```
